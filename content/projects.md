### Puzztool

[Puzztool](https://github.com/beckbria/puzztool) is a toolkit designed for solving puzzles, commonly used in competitions such as the MIT Mystery Hunt, Microsoft Puzzle Hunt, and Puzzled Pint.

It provides various features for encoding and decoding information, including:

*   **Encodings:** Tools for mapping numbers to letters (ASCII, Binary, Ordinal), Base64, Braille, Morse Code, Phone Keypad, Pigpen, and Semaphore.
*   **Ciphers:** Solvers for cryptographic methods such as Autokey, Caesar, and Vigenere.
*   **Reference:** Miscellaneous tools for NATO Alphabet, Naval Flags, resistor color codes, and character encodings.

Written in TypeScript/Node.js, the project is available as an open-source tool on [GitHub](https://github.com/beckbria/puzztool).
