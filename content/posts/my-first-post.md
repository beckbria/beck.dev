This post is compiled inside a Docker container using a recent stable version of Hugo (v0.128.2) and the beautiful **Congo** theme.

## Features

- **No local installation needed**: All dependencies are encapsulated in Docker.
- **Easy deployment**: Simply run the build script to generate static files in the local `public` folder.
- **Modern looks**: Congo theme provides sleek dark mode, accessibility, and high performance out-of-the-box.
