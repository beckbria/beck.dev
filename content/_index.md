Hello! Welcome to my Hugo static site, running entirely inside a Docker container.

### Getting Started

You can customize this site by modifying files in:
- `config/_default/params.toml` - for author links, bio, and styling options.
- `content/` - for posts and other markdown files.
- `assets/` - for images (like the profile picture).

Happy writing!
